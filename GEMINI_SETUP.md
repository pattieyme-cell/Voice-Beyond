### Step 1: Get a Valid Gemini API Key
1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy your new API key

### Step 2: Update the API Key
1. Open the file: `C:\voice beyond\.env`
2. Replace the current key with your valid key:
   ```
   GEMINI_API_KEY=YOUR_ACTUAL_API_KEY_HERE
   ```
3. Save the file

### Step 3: Restart the Backend
1. Close the current backend server (Ctrl+C in the terminal)
2. Run: `py app.py` from the Voice Beyond directory
3. The system will now use live Gemini AI responses!
